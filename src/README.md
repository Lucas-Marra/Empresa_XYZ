#Projeto Empresa XYZ

###Um programa feito em java que representa um sistema simples de uma empresa que cadastra os funcionários.



#####Tipos de funcionários:
* Presidente
* Gerente
* Secretário(a)
* Estagiário(a)

#####Funções do sistema:
* Adicionar funcionário
* Pesquisar pagamento de funcionário pelo ID
* Listar funcionários
